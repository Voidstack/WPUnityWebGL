Affiche un jeu Unity WebGL via un shortcode et permet d’uploader le build depuis l’administration WordPress.

== Description ==

Ce plugin WordPress permet de téléverser facilement un build Unity WebGL au format ZIP via une interface d’administration.  
Le build est automatiquement extrait dans le dossier `wp-content/uploads/unity-webgl/`.  
Un shortcode `[mon_jeu_unity]` permet d’afficher le jeu dans une iframe sur n’importe quelle page ou article.

== Installation ==

1. Téléversez le dossier du plugin dans le répertoire `/wp-content/plugins/` ou utilisez l’option "Téléverser une extension" dans l’admin WordPress.  
2. Activez le plugin via le menu "Extensions".  
3. Dans le menu WordPress, rendez-vous sur "Unity WebGL" pour uploader votre build Unity (fichier ZIP).  
4. Utilisez le shortcode `[mon_jeu_unity]` pour afficher le jeu.
